# База данных для MMORPG 

## Описание проекта
Данный проект представляет собой проектирование базы данных для многопользовательской ролевой онлайн-игры (MMORPG). База спроектирована с соблюдением принципов нормализации, включая нормальную форму Бойса-Кодда (НФБК).

## Основные сущности
1. **Игроки** - информация о пользователях
2. **Персонажи** - игровые персонажи, созданные игроками
3. **Классы** - доступные классы персонажей
4. **Расы** - доступные расы персонажей
5. **Предметы** - игровые предметы (оружие, броня и т.д.)
6. **Инвентарь** - связь персонажей с предметами
7. **Навыки** - умения и способности
8. **Зоны** - игровые локации
9. **Гильдии** - объединения игроков
10. **Квесты** - задания для выполнения
11. **Прогресс квестов** - отслеживание выполнения квестов

## Нормализация
База данных соответствует следующим нормальным формам:
- **1НФ**: Все атрибуты атомарны, нет повторяющихся групп
- **2НФ**: Нет частичных зависимостей от составных ключей
- **3НФ**: Нет транзитивных зависимостей
- **НФБК**: Каждый детерминант является потенциальным ключом
